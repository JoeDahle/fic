## NavButton Component
___

Simple navigation button

### Example
```js
<NavButton text={string} path={string} />
```

### Props
|Prop       | Type      | Default     | Possible Values
|-----------|-----------|-------------|-----------------------
|*text*     | string    | Go Home     |
|*path*     | string    | /home       |

##### Note
All path values **MUST** correspond to a route path in `routes.js`, and start with `/`.
